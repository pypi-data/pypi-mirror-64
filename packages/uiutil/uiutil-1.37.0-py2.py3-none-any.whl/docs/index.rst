Uiutil: For Easier TTK
======================

Uiutil sits on top of ttk/tkinter.
Its raison d'etre is to hide away as much ttk boiler plate as possible.
In order to do that, we made some decisions, like arbitrarily choosing
grid over pack, because that's what all our existing code used.
